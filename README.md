# Dreamhack Bot

드림핵 사이트의 강의와 퀴즈를 자동으로 순회하고 수료 처리하는 봇입니다.

## 주요 기능

- 🚀 **자동 강의 수강**: 커리큘럼에서 완료되지 않은 강의를 찾아서 자동으로 학습 과정을 수행합니다.
- 📝 **자동 퀴즈 풀이**: 브루트포스 방식으로 퀴즈를 자동으로 해결합니다. 오답 캐싱을 통해 풀이 속도를 최적화했습니다.
- 🤖 **사람과 같은 행동 패턴**: 계정 정지(블락) 방지를 위해 타이핑, 스크롤, 대기 시간 등을 사람처럼 무작위로 동작하게 합니다 (`puppeteer-extra-plugin-stealth` 및 무작위 딜레이 적용).
- 🍪 **세션 유지**: 사용자의 기본 로컬 크롬(Chrome) 프로필을 연동하여, 매 실행마다 번거롭게 다시 로그인할 필요가 없습니다.

## 요구 사항

- **Node.js**: v18 이상 권장
- **Google Chrome**: 로컬에 Chrome이 설치되어 있어야 합니다.

## 설치 및 실행

1. 저장소 클론 및 의존성 설치

   ```bash
   npm install
   ```

2. `.env` 파일 설정 (선택 사항)
   - `.env.example` 파일을 복사하여 `.env`로 변경합니다.
   - 크롬 경로(`CHROME_PATH`)나 타겟 커리큘럼 URL(`CURRICULUM_URL`)을 환경에 맞게 수정합니다.

3. 봇 실행
   ```bash
   npm start
   ```

## 구조 및 설정

- `bot.js`: 봇의 진입점(Entry point)이자 전체 파이프라인(순회)을 오케스트레이션합니다.
- `config.js`: 각종 셀렉터, 딜레이 시간(min/max), 환경변수 로딩 등을 관리합니다.
- `utils.js`: 브라우저 실행, 로그인 상태 대기, 랜덤 대기, 스크롤 등 공통 유틸리티 함수 모음입니다.
- `quizManager.js`: 퀴즈 페이지를 감지하고, 브루트포스(오답 캐싱 포함)로 문제를 푸는 로직을 담당합니다.
- `lectureManager.js`: 일반 강의 페이지에 머물면서 스크롤하고, 완료/다음 버튼을 누르는 로직을 담당합니다.
- `logger.js`: `winston` 기반의 로깅 시스템으로, 파일(`logs/bot.log`, `logs/error.log`)과 콘솔에 동시에 로그를 남깁니다.

## 개발 도구

- 코드 린팅: `npm run lint`
- 코드 포매팅: `npm run format`
